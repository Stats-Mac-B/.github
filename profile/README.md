## Overview

Stats Advanced is a free, open-source system monitoring application that displays comprehensive real-time metrics for CPU, GPU, RAM, disk, network, battery, and hardware sensors directly in the menu bar, providing the most complete hardware monitoring available without any cost.

The modular widget system lets you configure exactly which metrics appear in the menu bar and in what format, choosing between graphs, percentages, or custom displays that balance information density with visual clarity for your specific setup.

Deep hardware sensor access surfaces data that most monitoring tools do not expose, including individual core temperatures, voltage readings, power draw measurements, and fan speeds that give a complete picture of thermal and electrical hardware state.

Open-source development means the application is continuously improved by the community, with support for new hardware, new sensor types, and new display options added in response to real user needs without commercial priorities distorting development direction.

For users who want comprehensive, trustworthy system monitoring without cost or privacy concerns, Stats Advanced is the monitoring tool that the open-source community has built and maintains as the definitive free alternative to commercial system monitors.

## Performance

Stats balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management reduces overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance compared to software-only solutions. On Intel machines, hardware acceleration is used wherever available.
